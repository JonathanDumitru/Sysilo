# Personas

## Intent

Define the primary user personas, their goals, and the surfaces they use most.

## Personas

| Persona | Primary Focus | Key Surfaces |
| --- | --- | --- |
| Integration Developers | Build, test, deploy integrations | Integration Studio, CLI, IDE plugins |
| Platform and Ops Engineers | Monitor health, manage incidents, maintain agents | Operations Center, Agent management |
| Enterprise Architects | Define standards, govern integrations, drive rationalization | Governance Center, Asset Registry |

## Shared capabilities

- AI assistant with knowledge graph context
- Global search across integrations and assets
- Audit history and change tracking

## Open questions

- Which persona is the initial buyer vs daily user?
- What workflows must be supported offline or via CLI?
