# Sysilo Documentation

This is the working documentation set derived from the platform plan at
`docs/plans/2026-02-03-sysilo-platform-design.md`.

## How to use this set

- This is living documentation. Update sections as implementation decisions land.
- Each doc starts with intent and scope, then diagrams, then open questions.
- Diagrams use Mermaid so they can render in GitHub and many editors.
